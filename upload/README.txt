ESTRUTURA DE PASTAS DE PROJETOS
================================

Esta pasta contém 10 pastas de projetos (projeto1 a projeto10).

COMO USAR:
----------
1. Coloque até 5 imagens em cada pasta de projeto
2. Edite o arquivo projetos.json para configurar os detalhes de cada projeto
3. As imagens serão carregadas automaticamente pelo sistema

FORMATO DAS IMAGENS:
--------------------
- Formatos aceitos: JPG, PNG, GIF, WebP
- Nomeie as imagens como: imagem1.jpg, imagem2.jpg, etc.
- Ou use qualquer nome, mas configure no projetos.json

EXEMPLO DE ESTRUTURA:
---------------------
upload/
  ├── projetos.json
  ├── projeto1/
  │   ├── imagem1.jpg
  │   ├── imagem2.jpg
  │   ├── imagem3.jpg
  │   ├── imagem4.jpg
  │   └── imagem5.jpg
  ├── projeto2/
  │   └── ...
  └── ...
