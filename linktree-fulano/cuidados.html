Cuidados Especiais

Cuidados com a Pele

  • Limpe a pele duas vezes ao dia com um produto adequado para seu tipo de pele
  • Use protetor solar diariamente, mesmo em dias nublados
  • Mantenha a pele hidratada com produtos específicos para seu tipo
  • Faça esfoliação 1-2 vezes por semana para renovar as células
  • Nunca durma com maquiagem - sempre remova completamente
  • Beba bastante água para manter a hidratação de dentro para fora
  • Evite tocar o rosto com as mãos sujas

Cuidados com as Sobrancelhas

  • Não retire pelos em excesso - deixe para profissionais
  • Use um sérum para crescimento para manter os pelos saudáveis
  • Penteie diariamente com uma escovinha para estimular a circulação
  • Evite tinturas caseiras - produtos inadequados podem causar alergias
  • Hidrate com óleos naturais como rícino ou óleo de amêndoas
  • Não force a retirada de pelos encravados - procure um profissional
  • Proteja do sol com protetor solar ou chapéu em exposição prolongada

Cuidados com os Cílios

  • Remova a máscara de cílios todas as noites com demaquilante suave
  • Use um removedor oil-free se tiver extensões de cílios
  • Evite esfregar os olhos para não quebrar os cílios
  • Aplique sérum para cílios à noite para fortalecer
  • Não use máscara de cílios vencida - pode causar irritação
  • Para quem tem extensões: evite óleo na região dos olhos
  • Durma de lado ou de costas para não pressionar os cílios
  • Agende o retoque de extensões a cada 2-3 semanas
